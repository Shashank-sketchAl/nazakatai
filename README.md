# NazakatAI

## Team Name
TechieWarriors

## Problem Statement
PS-17 — AI stylist agent for Lucknow ethnic wear brands



## Project Idea
NazakatAI is an AI-powered fashion stylist platform designed for Lucknow ethnic wear brands. The platform helps users discover personalized Chikankari, Zardozi, and Nawabi fashion recommendations based on occasion, budget, and style preferences using conversational AI.

The goal is to modernize Lucknow’s traditional ethnic fashion ecosystem through AI-driven personalization, premium digital experiences, and artisan storytelling.
